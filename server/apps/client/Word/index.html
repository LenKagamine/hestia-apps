Word Building

Each player takes turns adding a letter onto a string of letters.

The objective is to add a letter such that the resulting string is not a valid word, but is a prefix of a valid word.

If you think someone has made a real word, or the string cannot be a real prefix, then you can call them out.

If you get called out, you have 10 seconds to enter your target word (if you did have a word). Otherwise, you will lose.
